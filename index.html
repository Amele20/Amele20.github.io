Web programming fundamentals : HTML et CSS



Tout d'abord le HTML est un langage de balise utilisé pour structurer et donner du sens à la page web, nous pouvons par exemple crée des paragraphes, ajouter des titres ou même des images.
Ensuite nous avons vu le CSS qui est un peu plus compliqué je trouve personellement. Le CSS lui est un langage de règles de style, c'est à dire pour mettre en forme le contenu, modifier la police, la taille etla couleur.
