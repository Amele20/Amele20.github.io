HTML & CSS

HTML et CSS sont des langages informatiques qui nous permettent donc de créer des sites web.Ces deux langages constituent des incontournables universels et pour tous les sites, constituent donc la base de tout site web. Votre navigateur se voit comme un traducteur de ce que vous voyez, de ce qui s'affiche, les différentes pages de votre site web. Chaque page web est désignée par son URL (Uniform Ressource Locator): WWW.estiam.com,www.google.fr, ...ect.
